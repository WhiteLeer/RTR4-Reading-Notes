## Chapter 9 - Physically Based Shading 基于物理的着色

> [!important] 本章核心
> 基于物理的着色（PBS）是现代渲染的基础，通过物理原理建立光与材质交互的数学模型，实现真实感渲染。

### 9.1 光的物理学

#### 光的波动性质

光是电磁横波：
- **电场和磁场**：垂直振荡，相互耦合
- **波长λ**：决定颜色（400-700nm可见光）
- **相位速度**：真空中约30万km/s

#### 干涉现象

光波叠加的三种情况：

**相长干涉**（Constructive Interference）：
- 波峰对齐，振幅增强
- n个波叠加 → n²倍强度

**相消干涉**（Destructive Interference）：
- 波峰波谷抵消
- 总强度为0

**不相干叠加**（Incoherent Addition）：
- 相位随机
- n个波叠加 → n倍强度（线性）

> [!tip] 能量守恒
> 干涉现象看似违反能量守恒，但在空间中整体是守恒的——相长处增强，相消处减弱。

#### 散射类型

**粒子散射**：
- **瑞利散射**（Rayleigh）：粒子<波长，波长依赖性强（天空蓝）
- **米氏散射**（Mie）：粒子>波长，波长依赖性弱（云雾白）

**介质中的光传播**：
- **折射率n**：相位速度变化的度量
- **衰减率κ**：吸收导致的振幅下降
- **复折射率**：n + iκ

**表面反射与折射**：
- 遵循菲涅尔方程
- Snell定律：n₁sin(θᵢ) = n₂sin(θₜ)
- 反射波和透射波的能量分配

#### 次表面散射

光在物体内部的行为：
- **局部SSS**：进出距离 < 像素大小（可用BRDF模拟）
- **全局SSS**：进出距离 > 像素大小（需要特殊技术）
- 金属：高吸收，无明显SSS
- 电介质：散射+吸收组合

### 9.2 相机模型

#### 针孔相机

理想化模型：
- 无限小光圈
- 每个传感器点对应一条光线
- 全局清晰，无景深

#### 镜头相机

更真实的模型：
- 有限光圈+透镜
- 景深效果（Depth of Field）
- 每个像素接收锥形区域光线

> [!note] 实时渲染简化
> 通常使用针孔模型，在后处理阶段模拟景深。

### 9.3 BRDF理论

#### BRDF定义

双向反射分布函数（Bidirectional Reflectance Distribution Function）：
```
f(l, v)：描述光从方向l入射，向方向v出射的反射特性
```

**参数**：
- **各向同性BRDF**：3个参数（θᵢ, θₒ, φ相对方位角）
- **各向异性BRDF**：4个参数（增加φᵢ或φₒ）

#### 反射方程

计算出射radiance：
```
Lₒ(v) = ∫_Ω f(l,v) Lᵢ(l) (n·l) dl
```

物理约束：
- **Helmholtz互易性**：f(l,v) = f(v,l)
- **能量守恒**：出射≤入射

#### 定向反照率

衡量能量损失：
```
R(l) = ∫_Ω f(l,v) (n·v) dv
```
- R(l) ∈ [0,1]
- 能量守恒要求：对所有l，R(l) ≤ 1

#### Lambertian BRDF

最简单的模型：
```
f = ρₛₛ/π
```
- 常数BRDF（完全漫反射）
- ρₛₛ：次表面反照率
- 1/π：归一化因子

---

### 9.4 光照模型

#### 局部vs全局光照

**局部光照**：
- Lᵢ(l)已知，只计算单点反射
- 适合实时渲染

**全局光照**：
- 递归计算光线传播
- 渲染方程的完整求解

#### 方向光近似

小且远的面光源→方向光：
```
Lₒ(v) = π f(lc, v) cₗᵢₘₕₜ (n·lc)₊
```

多光源：
```
Lₒ(v) = π Σᵢ f(lcᵢ, v) cₗᵢₘₕₜᵢ (n·lcᵢ)₊
```

### 9.5 菲涅尔反射

#### 外反射（n₁ < n₂）

特征：
- **法线入射**（θ=0°）：F₀（材质特征镜面色）
- **掠射角**（θ=90°）：趋向白色（1.0）
- 非线性增长曲线

#### Schlick近似

常用的高效近似：
```
F(n,l) ≈ F₀ + (1-F₀)(1-(n·l)₊)⁵
```

广义形式（艺术控制）：
```
F(n,l) ≈ F₀ + (F₉₀-F₀)(1-(n·l)₊)ⁿ
```

#### 典型F₀值

**电介质**（绝缘体）：
- 水：0.02
- 玻璃：0.04
- 塑料：0.03-0.05
- 宝石：0.05-0.17
- 特点：低F₀值（<0.06），无色反射

**金属**（导体）：
- 银：0.95-0.98
- 铝：0.91-0.92
- 金：1.00（R）, 0.82（G）, 0.37（B）
- 铜：0.95（R）, 0.64（G）, 0.54（B）
- 特点：高F₀值（>0.5），彩色反射

**半导体**：
- 硅：~0.30
- 介于电介质和金属之间
- 避免使用0.2-0.45范围（不真实）

> [!warning] 金属度工作流
> metallic参数化：用单个标量m控制
> - m=1：金属（F₀=cₛᵤᵣf, ρₛₛ=0）
> - m=0：电介质（F₀=常数, ρₛₛ=cₛᵤᵣf）

#### 内反射（n₁ > n₂）

特殊现象：
- **全内反射**：θᵢ > θc时，100%反射
- **临界角**：sin(θc) = n₂/n₁
- 常见于水下、玻璃内部

### 9.6 微观几何

#### 微表面法线分布

亚像素表面细节的统计描述：
- 每个像素包含大量微表面
- 各向同性：旋转对称
- 各向异性：有方向性（如拉丝金属）

#### 粗糙度影响

**表面粗糙度α**：
- α小：高光集中、镜面
- α大：高光扩散、粗糙

**视觉效果**：
- 反射环境细节模糊
- 高光范围变宽、颜色变暗
- 能量扩散到更大锥形角

#### Shadowing与Masking

**Shadowing**：
- 微表面遮挡光源
- 部分区域收不到光

**Masking**：
- 微表面遮挡相机
- 部分区域不可见

**逆反射效应**：
- 粗糙电介质的特征
- 光优先反射回入射方向
- 月球表面的视觉效果

#### 多次反弹

微表面间的相互反射：
- 金属：重要（唯一的漫反射来源）
- 电介质：相对较弱
- 有色金属：多次反弹颜色更深

### 9.7 微表面理论

#### 核心概念

将微观几何建模为微平面集合：
- **Micro-BRDF** fμ(l,v,m)：单个微平面的反射
- **NDF** D(m)：微平面法线分布
- **Masking** G₁(m,v)：可见性函数

#### 法线分布函数NDF

归一化约束：
```
∫_Θ D(m)(n·m)dm = 1
```

投影守恒：
```
∫_Θ D(m)(v·m)dm = v·n
```

#### Smith Masking函数

目前最佳选择：
```
G₁(m,v) = χ₊(m·v) / (1 + Λ(v))
```

特性：
- 法线-masking独立性
- 与NDF配合使用
- 每种NDF有对应的Λ函数

#### 联合Masking-Shadowing

考虑光线和视线的可见性：

**可分离形式**（过暗）：
```
G₂(l,v,m) = G₁(v,m) G₁(l,m)
```

**高度相关形式**（推荐）：
```
G₂(l,v,m) = 1 / (1 + Λ(v) + Λ(l))
```

#### 微表面BRDF推导

镜面micro-BRDF（完美菲涅尔镜面）：
```
fₛₚₑ𝒸 = F(h,l) G₂(l,v,h) D(h) / (4(n·l)(n·v))
```

关键：只有m=h的微表面贡献反射

### 9.8 表面反射的BRDF模型

#### 各向同性NDF

**Beckmann分布**：
```
D(m) = χ₊(n·m) exp(-(tan²θₘ)/αᵦ²) / (παᵦ²(n·m)⁴)
```
- 光学界经典模型
- αᵦ：粗糙度（0=完美镜面）
- 形状不变

**Blinn-Phong分布**：
```
D(m) = (αₚ+2)/(2π) (n·m)ᵅᵖ χ₊(n·m)
```
- 计算简单
- αₚ大→光滑；αₚ小→粗糙
- 常用映射：αₚ = mˢ（非线性）

**GGX分布**（Trowbridge-Reitz）：
```
D(m) = χ₊(n·m) αg² / (π((n·m)²(αg²-1)+1)²)
```
- 当前最流行
- 峰值窄，拖尾长
- 产生haze/glow效果
- 常用映射：αg = r²

**GGX vs Beckmann**：
- GGX：更真实的高光拖尾
- Beckmann：更对称的高光形状
- GGX更接近测量数据

**GTR分布**（广义TR）：
```
D(m) = k(α,γ) / ((n·m)²(α²-1)+1)^γ
```
- γ控制拖尾形状
- γ=2时退化为GGX
- 更多控制但更复杂

#### 各向异性NDF

扩展为椭圆形分布：
```
D(m) = g(√((m·t/αₓ)² + (m·b/αᵧ)²) / (n·m))
```
- αₓ、αᵧ：两个方向的粗糙度
- 需要切线空间（t, b, n）
- 适合拉丝金属、头发等

#### 多次反弹的高光

Imageworks方法：
```
fₘₛ = FₘₛRₛF₁(1-RₛF₁) / (π(1-Rₛ F₁R̄ₛF₁))
```
- 补偿微表面间多次反射
- 对粗糙金属尤其重要
- 预计算查找表实现

### 9.9 次表面散射的BRDF模型

#### 次表面反照率ρₛₛ

电介质的漫反射颜色：
- ρₛₛ ∈ [0,1]（RGB）
- 逃逸光/进入光的比值
- 散射vs吸收的竞争结果

**典型值**：
- 新雪：0.8+（高散射）
- 白漆：~0.7
- 混凝土：0.15-0.4
- 煤炭：~0.0
- 最低可达0.0035（OSHA Black）

> [!warning] 常见误解
> ρₛₛ下限并非0.03，这个值混合了镜面和漫反射。实际材质可以更暗。

#### 光滑vs粗糙表面

判断标准：**次表面散射距离vs表面不规则性**

**光滑表面模型**：
- 散射距离 > 不规则性
- 表面可视为平坦
- 无逆反射效应

**粗糙表面模型**：
- 散射距离 < 不规则性
- 需考虑微观几何
- 有逆反射效应（如月球）

#### 光滑表面漫反射模型

**简单Lambertian**：
```
fₐᵢff = ρₛₛ/π
```

**耦合漫反射**（Shirley）：
```
考虑表面-次表面能量权衡
fₐᵢff = (ρₛₛ/π)(1-F)(expression)
```

**Kelemen-Szirmay-Kalos**：
```
fₐᵢff = (ρₛₛ/π)(1-Rₛₚₑ𝒸)(1-R̄ₛₚₑ𝒸) / (1-R̄ₛₚₑ𝒸)
```
- 可与任何高光项配合
- 需要预计算查找表
- Rₛₚₑ𝒸：镜面定向反照率

#### 粗糙表面漫反射模型

**Disney漫反射**：
```
fₐᵢff = (ρₛₛ/π) fd
fd包含粗糙度依赖的缩放
```
- 基于测量数据拟合
- 电影和游戏广泛使用
- 包含光泽项用于织物

**Hammon微表面漫反射**：
```
基于GGX的micro-BRDF
考虑多次反弹
相对简单的拟合函数
```
- fₛₘₒₒₜₕ：光滑项
- fᵣₒᵤₘₕ：粗糙项
- fₘᵤₗₜᵢ：多次散射项

**Oren-Nayar**：
- 经典粗糙表面模型
- Lambertian micro-BRDF
- 球面高斯NDF
- 有简化的"定性"版本

### 9.10 布料的BRDF模型

#### 布料的独特性

与其他材质的差异：
- 高度重复的编织结构
- 突出的圆柱形纤维
- 各向异性高光
- 粗糙散射（边缘明亮）
- 颜色随角度变化

> [!tip] 高频细节重要性
> 除了BRDF，织物的空间变化（编织纹理、老化、皱纹）对真实感至关重要。

#### 经验布料模型

**边缘光照**（《神秘海域2》）：
```
边缘项 + 内部项 + Lambertian
kᵣᵢₘ(1-n·v)^αᵣᵢₘ + kᵢₙₙₑᵣ(n·v)^αᵢₙₙₑᵣ + kdiff
```

**环绕光照**（《神秘海域4》）：
```
近似次表面散射
修改(n·l)的计算
```

**Disney光泽项**：
```
fₛₕₑₑₙ = kₛₕₑₑₙ cₛₕₑₑₙ (1-(h·l)₊)⁵
```
- 模拟粗糙散射
- 补偿能量损失

#### 微表面布料模型

**逆高斯NDF**（Ashikhmin天鹅绒）：
```
用于天鹅绒等材质
在掠射角有增强
```

**《教团1886》布料**：
- 改进的微表面BRDF
- 天鹅绒NDF一般化
- 支持各种粗糙纤维

**Imageworks光泽**：
```
不同的逆NDF
有Smith G2近似
能量守恒处理
```

#### 微圆柱体布料模型

**Kajiya-Kay BRDF**：
- 一维线段假设
- 法线平面定义
- 简单但有效
- 用于丝绸、天鹅绒

**Dreamworks模型**：
- 艺术家可控
- 纹理控制方向/粗糙度
- 支持经纬线不同参数
- 模拟变色织物

**头发模型的应用**：
- Marschner模型的R项
- PxrSurface的"模糊"波瓣
- 适用于某些布料类型

### 9.11 波动光学的BRDF模型

#### 几何光学vs波动光学

**几何光学假设**：
- 不规则性 << λ 或 >> 100λ
- 光建模为射线
- 忽略衍射和干涉

**波动光学需求**：
- 纳米几何：1-100λ范围
- 考虑光的波动性
- 衍射和薄膜干涉

#### 衍射模型

**Huygens-Fresnel原理**：
- 波前每点都是新波源
- 球面波相干叠加
- 解释衍射现象

**纳米几何效应**：
- 镜面反射 + 衍射光
- 不规则性↑ → 衍射角↓
- 周期结构 → 彩虹色

**实时应用**：
- Holzschuch模型：微表面+衍射
- Toisoul方法：周期性彩虹色
- 许多普通材质有显著衍射

#### 薄膜干涉模型

**物理原理**：
- 顶部和底部反射光干涉
- 路径长度差 → 相位差
- 波长选择性相长/相消

**相干长度**：
- 决定薄膜厚度阈值
- 白光：~1微米
- 激光：可达数英里

**常见效果**：
- 肥皂泡、油渍
- 金属涂层
- 皮革等普通材质的微妙色彩

**实时实现**：
- Smits-Meyer：1D查找表
- 考虑厚度、角度、IOR
- 预计算RGB颜色
- Belcour-Barla：更精确但更贵

### 9.12 分层材质

#### 透明涂层（Clear Coat）

最常见的分层案例：
- 光滑透明层 + 粗糙基底
- 二次反射效果
- 对金属基底最明显

**主要效果**：
1. 双层菲涅尔反射
2. 基于路径长度的吸收（Beer-Lambert）
3. 折射率变化

**工业应用**：
- Disney原则模型
- 虚幻引擎
- RenderMan PxrSurface
- Imageworks/Dreamworks

#### 通用分层系统

**逐层独立法线**：
- 水流过平面
- 冰覆盖粗糙土壤
- 皱巴巴的保鲜膜
- 实时应用中较少见

**Weidlich-Wilkie模型**：
- 任意数量层叠加
- 追踪反射/折射事件
- 假设层厚 << 微表面尺寸
- 不考虑层间多次反弹

**《使命召唤无限战争》**：
- 任意层数
- 折射、散射、吸收
- 逐层法线
- 60Hz实时运行

### 9.13 混合和过滤材质

#### 材质混合

**目的**：
- 组合多种材质属性
- 空间变化（纹理）
- 蒙版控制

**混合参数**：
- 高光颜色F₀
- 漫反射颜色ρₛₛ
- 粗糙度α
- 表面法线n

**应用场景**：
- 锈迹金属
- 动态伤害
- 用户自定义
- 环境多样性

**实现方式**：
- 预烘焙（离线）
- 实时混合（着色器）
- 参数插值 vs 结果插值

> [!warning] 非线性参数
> 粗糙度等非线性参数直接插值会有误差，但通常可接受。

#### 材质过滤问题

**根本原因**：
- GPU纹理过滤假设线性关系
- 但BRDF参数→颜色是非线性的
- 导致高光锯齿和外观变化

**瑕疵类型**：
1. **高光锯齿**：闪烁高光
2. **意外变化**：距离导致的光泽度/亮度改变

#### 过滤法线与NDF

**核心思想**：
- 法线贴图定义细观尺度
- NDF定义微观尺度
- mipmap降低分辨率 → 细观变微观

**理想方法**：
- 过滤NDF而非法线
- 保持方差信息
- 调整粗糙度参数

**Toksvig方法**：
```
利用平均法线长度反映方差
α' = αₚ√(∥n∥/(1-αₚ²(1-∥n∥)))
```
- 适用于Blinn-Phong
- 兼容GPU纹理过滤
- 需要未归一化法线

**LEAN/CLEAN映射**：
- 协方差矩阵方法
- 支持各向异性
- 适合动态法线
- 需要额外存储

**方差映射技术**：
- 预计算方差损失
- 存储在粗糙度贴图中
- 静态法线贴图优化
- 与压缩算法兼容

**增强技术**：
- 各向异性方差（xy方向）
- 几何曲率考虑（导数指令）
- 漫反射的n·l修正

#### 闪烁高光问题

**成因**：
- 像素内凸起数量不足
- 无法平滑平均
- 运动时帧间闪烁

**解决方案**：

**Wang-Bowles技术**：
- 稀疏闪光材质（雪花）
- 产生闪亮外观
- 时空稳定性

**Zirr-Kaplanyan方法**：
- 多尺度法线分布
- 更广泛材质支持
- 时空连续性

### 第九章核心要点总结

#### 物理基础

**光的双重性**：
```
波动性 → 干涉、衍射、薄膜干涉
粒子性 → 能量传递、散射
```

**材质分类**：
- **电介质**：低F₀（<0.06），有色ρₛₛ，无色镜面
- **金属**：高F₀（>0.5），彩色镜面，无ρₛₛ
- **半导体**：中等F₀（0.2-0.45），避免使用

#### BRDF核心公式

**反射方程**：
```
Lₒ(v) = ∫_Ω f(l,v) Lᵢ(l) (n·l) dl
```

**微表面BRDF**：
```
fₛₚₑ𝒸 = F(h,l) G₂(l,v,h) D(h) / (4(n·l)(n·v))
```

**组成要素**：
- **F**：菲涅尔反射（Schlick近似）
- **G₂**：Masking-Shadowing（Smith函数）
- **D**：NDF（GGX/Beckmann/Blinn-Phong）

#### 实用技术选择

**高光模型**：
```
推荐配置：
- NDF: GGX（真实拖尾）
- G₂: Smith高度相关
- F: Schlick近似
- 多次反弹：Imageworks方法
```

**漫反射模型**：
```
光滑表面：
- 简单：Lambertian
- 精确：Kelemen-Szirmay-Kalos

粗糙表面：
- Disney漫反射（广泛使用）
- Hammon微表面漫反射（理论严谨）
```

**特殊材质**：
- **布料**：微圆柱体模型或逆高斯NDF
- **薄膜**：干涉查找表
- **分层**：透明涂层+基底

#### 参数化方案

**金属度工作流**：
```
输入：cₛᵤᵣf（表面色），m（金属度）
输出：F₀和ρₛₛ

if m==1: F₀=cₛᵤᵣf, ρₛₛ=0  (金属)
if m==0: F₀=0.04, ρₛₛ=cₛᵤᵣf  (电介质)
```

**粗糙度映射**：
```
艺术参数：r ∈ [0,1]（线性感知）
技术参数：αg = r²（GGX）
```

#### 优化技术

**查找表**：
- Rₛₚₑ𝒸：镜面定向反照率（2D/3D）
- 薄膜干涉：路径长度差（1D RGB）
- 方差映射：法线方差（存储在α贴图）

**简化策略**：
- 多次反弹：仅次要贡献时可省略
- 各向异性：非必要时用各向同性
- 分层：简单透明涂层 vs 完整多层

#### 常见陷阱

**能量问题**：
- ❌ 直接相加漫反射+镜面
- ✅ 使用耦合项：(1-F)或(1-Rₛₚₑ𝒸)

**过滤错误**：
- ❌ 直接线性过滤粗糙度
- ✅ 使用方差映射技术

**参数范围**：
- F₀ ∈ [0.02, 1]（避免0.2-0.45除特殊情况）
- ρₛₛ ∈ [0, 1]（可低至0.0035）
- α > 0（避免除零）
